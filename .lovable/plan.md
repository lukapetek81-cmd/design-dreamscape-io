## AI Features Roadmap for Commodity Hub

You already have a strong commodities trading platform (real-time prices across 6 categories, portfolio, watchlists, price alerts, COT reports, term structure, volatility cone, roll scanner, spread/position calculators, expert insights, news, sentiment). Here's a prioritized set of AI features designed to serve retail traders, hedgers, and institutional users simultaneously.

### Tier 1 — Build first (highest impact, low complexity)

**1. AI Commodity Copilot (threaded chat assistant)**
A conversational assistant grounded in your own platform data. Users can ask:
- "Why is WTI down today?" → pulls live price + recent news + COT positioning
- "Summarize my portfolio risk" → reads their portfolio table
- "Compare Brent vs WTI term structure" → renders an inline chart
- "Set a price alert on gold if it breaks $2,400" → uses tool calling to create the alert

Built with AI SDK + Lovable AI Gateway (Gemini 3 Flash default), streamed via a Supabase edge function. Threaded conversations stored in Supabase with RLS. Tool calling lets the AI invoke your existing edge functions (fetch prices, create alerts, query portfolio, fetch news).

**2. AI News Digest & Impact Scoring**
Auto-generated daily/weekly briefing per user, scoped to their watchlist. Each news item gets an AI-assigned:
- Sentiment score (bullish/bearish/neutral)
- Impact score (1-5) on specific commodities
- One-sentence "why it matters"

Replaces noisy news scrolling with curated intelligence. Generated by a scheduled edge function.

**3. Smart Price Alert Suggestions**
When a user views a commodity, AI suggests technically meaningful alert levels based on recent volatility, support/resistance, term-structure inflection points, and COT positioning extremes — turning your existing alerts page into a guided experience.

### Tier 2 — Differentiators

**4. Portfolio AI Risk Analyst**
On the Portfolio page, an AI panel that:
- Detects concentration risk (e.g., "78% energy exposure")
- Surfaces correlated positions (uses your existing correlation data)
- Flags upcoming events from the Economic Calendar that could impact holdings
- Suggests hedges (e.g., "Consider a short WTI position to hedge airline exposure")

**5. Natural Language Market Screener**
Replace/augment the existing screener with: "Show me agricultural commodities in contango with bullish COT positioning and rising 30-day vol." AI parses intent and translates to filters.

**6. AI Trade Idea Generator (Premium)**
Daily AI-generated trade ideas (spreads, outright, calendar rolls) with reasoning, entry/exit/stop levels, and confidence score. Drawn from your COT, term structure, volatility, and correlation data. Strong premium upsell — gate behind the existing RevenueCat paywall.

### Tier 3 — Advanced (later)

**7. Chart Pattern Recognition (vision AI)**
User snapshots a chart → multimodal AI identifies patterns (head & shoulders, breakouts, divergences) and explains the setup. Uses Gemini multimodal.

**8. Hedger Mode — Physical Exposure Calculator**
For commodity buyers/sellers: input physical exposure ("I buy 10,000 bbl diesel/month"), AI recommends futures hedge ratio, contracts, and roll schedule.

**9. Voice-First Briefing (mobile)**
"Hey Copilot, what's my portfolio doing?" → 60-second spoken market summary on Android, using Gemini multimodal + TTS.

### Technical approach

```text
┌─────────────────────────────────────────────────────┐
│ Frontend: /copilot route                            │
│  - AI Elements (Conversation, Message, PromptInput) │
│  - Threaded sidebar (Supabase persistence)          │
└──────────────────┬──────────────────────────────────┘
                   │ streamText via DefaultChatTransport
┌──────────────────▼──────────────────────────────────┐
│ Edge Function: ai-copilot                           │
│  - AI SDK + Lovable Gateway (gemini-3-flash)        │
│  - JWT-authenticated, Zod-validated                 │
│  - Tool calling → invokes existing edge functions:  │
│    • fetch-commodity-prices                         │
│    • fetch-commodity-news                           │
│    • fetch-cot-report                               │
│    • fetch-forward-curve                            │
│    • query user's portfolio/watchlist/alerts (RLS)  │
│  - Persists messages to ai_threads/ai_messages      │
└─────────────────────────────────────────────────────┘
```

New tables (with proper GRANTs + RLS scoped to `auth.uid()`):
- `ai_threads` — id, user_id, title, created_at, updated_at
- `ai_messages` — id, thread_id, role, parts (jsonb UIMessage[]), created_at
- `ai_daily_digests` — id, user_id, date, content (jsonb), created_at

Premium gating: trade ideas (#6) and unlimited copilot messages behind existing RevenueCat Premium tier.
